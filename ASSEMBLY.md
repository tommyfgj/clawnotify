# 硬件装配说明（clawnotify）

> ESP32-C3 + LY011E 电磁铁 + Claude 像素头像的桌面敲击器。
> 通电瞬间，电磁铁吸住底部铁芯让整根推杆下拽，顶部 Claude 头向下一沉，断电弹回 →
> 桌面上听到"咚"的一声，作为 agent 的物理提醒。
>
> 本文档描述的是 clawnotify 使用的硬件（电路 + 机械 + 装配）。配套固件见
> `firmware/agent_notifier/`，主机端 CLI 见 `tools/notify.py`，整体架构见 [README](README.md)。

---

## 1. 工作原理

```
 [Claude 头 Invader]           ← 装饰，粘死在推杆顶端（内插 8mm + 502 胶）
         ║
         ║ 推杆 PushRod  φ3 × 70.8mm
         ║
         ║   ── 自由穿过 Cover 顶部 φ3.4 导向筒（Z=86~98） ──
         ║
         ║
  ┌──────╫──────┐
  │   [ClaudeFoot]   │ ← 铁芯 8×8×6mm（可动），粘死在推杆底端（内插 4mm + 502 胶）
  │   [电磁铁顶]     │
  │──────────────────│
  │  LY011E 电磁铁    │ ← 5V/1A 脉冲通电时向下吸住 Claude 脚
  │──────────────────│
  │  Base_Top 底板   │
  │──────────────────│
  │  (PCB 电气层)    │
  │  ESP32-C3 SuperMini │
  │  USB-C 朝后壁    │
  └──────────────────┘
```

**动作序列**（主机 `notify.py` 通过 USB 串口触发）：

1. 静止态：推杆重力自然上提，Claude 头高位
2. ESP32 收到串口指令（如 `TAP 60`）→ GPIO4 拉高 → MOS 导通 → 电磁铁通电
3. 电磁铁吸住 Claude 脚下拉 4mm → Claude 头"咚"一声下沉
4. ~60~100ms 后 GPIO4 拉低 → 电磁铁断电
5. 推杆 + Claude 头重力回弹 → 回到高位

## 3. 电路原理图

```
                  USB-C 5V IN
                       │
       ┌───────────────┤
       │               │
   [PPTC 1A]           │
       │               │
       ├───[C1 470µF]──┴── GND     ← 浪涌缓冲电容（靠近电磁铁）
       │
  +5V  ├───────── 电磁铁(+) 红线
       │                │
       │                │  (线圈)
       │                │
       │         电磁铁(-) 黑线 ─┬─── MOSFET Drain
       │                        │
       │                     [1N4148]      ← 续流（黑带朝 +）
       │                        │
       │                        └──────────→ +5V  (反并联)
       │
       ├── ESP32-C3 VBUS ── 板载 LDO → 3.3V
       │     │
       │     ├── GPIO4 ─────────→ MOSFET Gate
       │     ├── GND   ─────────→ MOSFET Source
       │     │
       │     └──[C2 100µF]── GND  ← 去耦
       │
      GND
```

**引脚分配**：

| ESP32-C3 引脚 | 功能 | 说明 |
|---|---|---|
| GPIO4 | 电磁铁驱动输出 | 推挽，高电平吸合 |
| GPIO8（板载 LED） | 状态指示 | 可选 |
| USB-C | 供电 + 烧录 | 5V/≥1A |

## 5. 装配步骤（5 步）

### 步骤 1 — 组装"吉祥物组件"（推杆 + 脚 + 头）

```
  [Claude 头 Invader] ← φ3.2×8 底孔
          ↓  涂 502 胶
    推杆 φ3 × 70.8 ← 插入 8mm
          ↓
    推杆底端
          ↓  涂 502 胶
    [Claude 脚] ← φ3.2×4 顶孔，插入 4mm
```

1. 把 φ3 推杆一端插入 Claude 头底部 φ3.2 孔，**深度 8mm**（到底），点 502 固定
2. 另一端插入 Claude 脚顶面 φ3.2 孔，**深度 4mm**，点 502 固定
3. 等胶水固化 1~2 小时

### 步骤 2 — 焊接电气（壳外完成并测试）

1. **去 USB-C**：拿 ESP32-C3 SuperMini，注意 USB-C 在短边（18.5mm 边），不能焊错方向
2. **焊接按顺序**：
   - ESP32 `5V` → MOS 板 `VIN`（用红杜邦线）
   - ESP32 `GND` → MOS 板 `GND`（黑线）
   - ESP32 `GPIO4` → MOS 板 `IN`（黄线）
   - MOS 板 `OUT+` → 电磁铁红线
   - MOS 板 `OUT-` → 电磁铁黑线
   - 1N4148 反并联电磁铁两端（黑带朝红）
   - **并联 C1（470µF）** 在电磁铁红-黑线之间（电解电容负极接黑）
   - （可选）PPTC 串在电磁铁红线回路
3. **测试**：USB-C 上电，上传 Arduino 固件（见第 6 节），串口触发一次 → 听到电磁铁"咔"声 ✓

### 步骤 3 — 装 Base_Bottom

1. 拎起 Base_Bottom，顶面朝上（开口朝上）
2. 把已焊好按钮的 ESP32-C3 从顶部 **垂直按下**
   - PCB 长边（22mm）对齐 Y 方向，USB-C 朝后壁 -Y 方向
   - 4 角对准 4 根托脚
   - 碰到左右倒钩时 **继续下压**（倒钩因割缝会弹性外扩）
   - 听到"咔"一声 → PCB 到位（底面贴托脚，顶面被倒钩压住）
3. 检查 USB-C 母座对准后壁 10×4.4mm 开孔 ✓
4. **把电磁铁 2P 线从 Base_Top 后方缺口预留**（暂时留 30mm 长度）
5. MOS 板用双面胶贴在 PCB 空余位置或侧壁内

### 步骤 4 — 装 Base_Top + 电磁铁

1. 电磁铁从 **Base_Top 的电磁铁卡托** 上方插入：
   - 底部压入卡托（10.2×8.5 内腔，间隙 0.2mm）
   - 出轴朝上，从 Base_Top 中央 φ4.5 孔伸出
   - 2P 线从卡托后方 3mm 缺口穿出 → 穿过 Base_Top 底板（若是从 Base_Top 下方穿过则需预先布线）
2. **涂胶**：在 Base_Top 底板**底面边缘一圈** 涂 502 / AB 胶（不要涂到电磁铁出轴孔附近）
3. **对齐压合**：Base_Top 和 Base_Bottom 外形一致（28×28），边缘对齐压紧
4. 按住 30 秒 → 静置 24h 固化

### 步骤 5 — 装吉祥物组件 + 外壳

1. 把"吉祥物组件"倒置：Claude 脚朝下
2. **Claude 脚放到电磁铁出轴顶上**（自然吸附 / 轻轻放）
3. **套外壳**：
   - 拿 Cover，底面朝下（开口朝下）
   - 对准底座和推杆
   - **推杆顶端先穿入 Cover 顶部 φ3.4 导向筒**
   - 继续下压，Cover 底部 U 型缺口对准 USB-C 位置
   - Cover 底边贴桌面
4. **拧 M2 侧螺丝**：从 Cover 左右侧 φ3.8 埋头孔 → 穿过 Cover 侧壁 → 拧进 Base_Bottom 侧壁的 M2 自攻预留孔
5. 整机完成 ✓

## 7. 电路保护（防 USB-C 烧坏）

LY011E 吸合瞬间浪涌可达 1.2~1.5A，长期会让 USB-C 焊盘发热甚至脱焊。推荐组合防护：

| # | 元件 | 作用 | 必须？ |
|---|---|---|---|
| C1 | 470µF 电解 | 吸收浪涌，USB-C 只需提供平均电流 | **必加** ★★★ |
| C2 | 100µF 陶瓷/钽 | 稳定 ESP32 VBUS 电压 | **必加** ★★★ |
| PPTC | 1A/2A 自恢复 | 过流自断，冷却恢复 | 推荐 ★★ |
| 固件 | PULSE ≤100ms + COOLDOWN ≥500ms | 软件限时 | **必加** ★★★ |
| TVS | SMAJ5.0A | 抑制瞬态过压 | 选配 ★ |

## 8. 安全须知

- ⚠️ **LY011E 是瞬时通电电磁铁**，单次通电**不得超过 1000ms**，否则线圈烧毁
- ⚠️ **必须加 1N4148 续流**，否则断电瞬间的反电动势会击穿 MOS
- ⚠️ **USB-C 电源必须 ≥1A**，否则吸合瞬间 ESP32 欠压复位
- ⚠️ **Claude 脚必须用铁**（304 不锈钢不吸）
- ⚠️ 外壳开孔尺寸已预留装配间隙，**不要修改缩放**后直接打印

## 10. 常见问题 FAQ

**Q1：Claude 头弹不动？**
- 电磁铁力太小？检查是否 5V 1A 版本（不要买 3V 的）
- 推杆太重？换铝/黄铜
- Claude 脚不吸？必须是铁材

**Q2：ESP32 一触发就重启？**
- USB-C 电源不够 1A → 换电源
- 没加 C1 470µF 电容 → 加上

**Q3：PCB 卡不进 Base_Bottom？**
- 打印公差大？用刀片轻刮挡边内侧 0.1mm
- 倒钩太硬？用热风枪轻吹挡边顶部 3 秒软化后按入

**Q4：Base_Top 和 Base_Bottom 粘不牢？**
- 先用砂纸 240 目磨粗结合面再上 AB 胶
- 用 502 不够强 → 换环氧 AB 胶

**Q5：推杆在外壳内歪斜/卡滞？**
- 推杆弯？换直的金属棒
- 导向筒内径太紧？稍微扩一下（用 φ3.5 钻头手工铰）
