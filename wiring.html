工作时序 · 上电后周期自动触发

参数可调：ACTION_MS（吸合时长，≤3000ms）· COOLDOWN_MS（间隔时长）

对应固件（Arduino / ESP32-C3）

// Clawbot - ESP32-C3 + LY011E，无按钮，周期自动触发
const int PIN_MOS = 4;              // 驱动电磁铁

const unsigned long ACTION_MS   = 500;    // 电磁铁吸合时长 (≤ 3000)
const unsigned long COOLDOWN_MS = 2500;   // 两次之间的冷却/间隔

void setup() {
  pinMode(PIN_MOS, OUTPUT);
  digitalWrite(PIN_MOS, LOW);
}

void loop() {
  digitalWrite(PIN_MOS, HIGH);   // 吸合
  delay(ACTION_MS);
  digitalWrite(PIN_MOS, LOW);    // 释放
  delay(COOLDOWN_MS);              // 冷却
}

可选加固 · 在 5V/GND 并联电容（防 ESP32 复位）

⚠️ 电磁铁吸合时长 ≤ 3s · 必须保留 1N4148 续流 · USB 供电 ≥1A
